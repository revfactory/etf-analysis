# Portfolio Analysis

ETF 포트폴리오의 시계열 비중 변화 분석과 인사이트 도출 방법론.

## 분석 절차

### 1. 기초 정규화
- 모든 시점의 상위 10 + 상위 20 종목 티커 집합을 union하여 분석 유니버스 구성
- 각 시점에서 해당 유니버스의 종목 비중을 나란히 정렬
- 티커 리네이밍은 이력 매핑으로 통합

### 2. 분류 (ETF별)

| 분류 | 기준 |
|------|------|
| **신규 편입 (New)** | 이전 시점 비중 0%, 최신 시점 비중 > 0.1% |
| **완전 제외 (Removed)** | 이전 시점 비중 > 0.1%, 최신 시점 비중 0% |
| **증액 (Increased)** | Δ 비중 ≥ +0.5%p |
| **축소 (Decreased)** | Δ 비중 ≤ -0.5%p |
| **유지 (Stable)** | |Δ 비중| < 0.5%p |

### 3. 집중도 지표

- **Top10 Concentration** = 상위 10 종목 비중 합 (%)
- **섹터 HHI** = Σ(섹터별 비중)² × 10000 (허핀달 지수)
- **유효 종목 수 (Effective N)** = 1 / Σ(비중²) — 실질 분산도

### 4. 크로스 ETF 비교

- **공통 편입 종목** — 3개 ETF 모두에 편입 (교집합)
- **단독 편입 종목** — 1개 ETF에만 편입 (단일 등장)
- **비중 차이 큰 종목** — 동일 종목의 ETF 간 비중 차이 ≥ 2%p

### 5. 인사이트 도출 (3~5개)

좋은 인사이트는 데이터에 기반하고 실행 가능한 함의를 준다:
- "X 섹터 편입 비중이 최근 6개월간 평균 +3.2%p 증가" (트렌드)
- "KODEX는 Y 종목을 완전 제외, TIGER는 반대로 증액 — 전략 방향성 차이" (대비)
- "상위 3개 모두 알테오젠 편입 상위 3위 내 — 지수 영향력 집중" (리스크)

### `02_insights.md`
자연어 인사이트 3~5개. 각 인사이트는 "헤드라인 문장 + 수치 근거 + 함의" 3줄 구조.

### `02_charts_spec.json`
시각화용 표준 데이터. web-designer와 pdf-author 공용.

차트 종류:
1. `top10_weight_comparison` — ETF별 상위 10종목 그룹 막대
2. `weight_change_trend` — 주요 종목 비중 시계열 라인
3. `classification_breakdown` — 신규/증액/축소/제외 도넛
4. `sector_heatmap` — ETF × 섹터 히트맵
5. `common_vs_unique` — 공통/단독 편입 벤 다이어그램 데이터

## 분석 원칙

- **유의미한 변화만 강조** — ±0.5%p 미만은 노이즈
- **이벤트 주석** — 상장, 분할, 합병 등 구조적 이벤트는 표기
- **상대 비교 가능하게** — 절대값과 함께 변화율/상대비중도 병기
- **한계 명시** — 시점이 2개 이하이면 트렌드 결론 금지
